# افسانه‌ها 🏔️

گنجینه‌ی مردمی و متن‌باز افسانه‌ها و قصه‌های محلی ایران. هر کسی — حتی بدون حساب گیت‌هاب — می‌تواند افسانه‌ی شهر و روستای خود را ثبت کند. نام نویسنده همیشه حفظ می‌شود.

> English README: ./README.md

## راه‌اندازی سریع (فقط چند دقیقه)

این سایت کاملاً **ایستاتیک** است و هیچ مرحله‌ی بیلدی لازم ندارد.

1. محتوای زیپ را در ریشه‌ی یک مخزن گیت‌هاب اکسترکت کن (مثلاً مخزنی به نام `afsaneha`).
2. در **Settings ← Pages** انتشار را روی شاخه‌ی `main` و پوشه‌ی `/ (root)` بگذار.
3. تمام! سایت روی `https://USERNAME.github.io/afsaneha/` بالا می‌آید.

فایل `.nojekyll` و `assets/data/legends.json` از پیش آماده‌اند، پس بدون هیچ کار اضافه‌ای کار می‌کند.

> فهرست افسانه‌ها به‌صورت خودکار هم ساخته می‌شود: ورک‌فلوی `.github/workflows/deploy.yml` از پیش تنظیم شده و در هر push فایل `legends.json` را از روی پوشه‌ی `legends/` می‌سازد و سایت را منتشر می‌کند.

## ساختار پروژه

```
index.html          صفحه‌ی اول (دکمه‌ی فارسی/انگلیسی + دارک/لایت + جستجو و فیلتر)
legend.html         صفحه‌ی نمایش یک افسانه
contribute.html     فرم ثبت افسانه‌ی جدید
about.html, 404.html
config.json         تنها جایی که آدرس Cloudflare Worker را می‌گذاری
assets/
  css/style.css
  js/            منطق سایت (i18n, md, app, legend, contribute)
  img/           همه‌ی لایه‌های طراحی + نسخه‌های نام‌گذاری‌شده (logo/map/bg...)
  data/legends.json   فهرست خودکارِ افسانه‌ها
legends/            ⬅️ فقط متن! هر افسانه یک فایل markdown تمیز
  <استان>/<slug>/fa.md  (و en.md اگر نسخه‌ی انگلیسی موجود باشد)
worker/             ربات Cloudflare (جدا از سایت، منتشر نمی‌شود)
scripts/build-index.mjs   سازنده‌ی فهرست
.github/workflows/deploy.yml
```

> پوشه‌ی `legends/` عمداً **بدون هیچ کد جاوااسکریپتی** است؛ فقط متن مرتب با مقداری متادیتا در بالای فایل. هر کسی می‌تواند تنها به همین پوشه تکیه کند و یک آرشیو کامل و خوانا داشته باشد.

## زبان (فارسی/انگلیسی)

- **صفحه‌ی اول و کل رابط** دو زبانه است؛ دکمه‌ی گوشه‌ی بالا زبان را عوض می‌کند.
- **افسانه‌ها** فقط اگر نویسنده نسخه‌ی انگلیسی (`en.md`) گذاشته باشد، دو زبانه نمایش داده می‌شوند. اگر نگذاشته باشد، اجباری نیست و سایت یک یادداشت کوتاه نشان می‌دهد و متن فارسی را نمایش می‌دهد.

## تم دارک/لایت

دکمه‌ی ◐ بین تم شب (نقشه‌ی تیره) و روز (نقشه‌ی پارچمنت) جابه‌جا می‌کند؛ انتخاب در مرورگر ذخیره می‌شود.

## مشارکت بدون حساب گیت‌هاب

فرم «ثبت افسانه» داده را به Cloudflare Worker می‌فرستد، Worker یک Pull Request روی همین مخزن باز می‌کند و تو پیش از مرج بازبینی می‌کنی. راه‌اندازی ربات در `worker/README.md` توضیح داده شده. آدرس Worker را فقط در `config.json` بگذار.

### ⚠️ چون مخزن عمومی است
- **هیچ توکنی داخل مخزن نگذار.** توکن فقط در Worker Secrets می‌ماند.
- ترجیحاً به‌جای توکن شخصی از **GitHub App** یا fine-grained PAT محدود به همین یک مخزن استفاده کن.
- **Turnstile (کپچا)** را روی فرم فعال کن تا اسپم ربات‌ها جلوگیری شود.
