# مشارکت در افسانه‌ها | Contributing

دو راه برای افزودن افسانه وجود دارد:

## ۱) از طریق سایت (بدون نیاز به گیت‌هاب)
به صفحه‌ی «ثبت افسانه‌ی جدید» برو، فرم را پر کن و بفرست. یک Pull Request خودکار ساخته می‌شود و پس از بازبینی منتشر می‌شود.

## ۲) مستقیم در مخزن (برای کاربران گیت‌هاب)
یک فایل جدید بساز:

```
legends/<استان>/<slug>/fa.md
```

قالب فایل (frontmatter + متن):

```markdown
---
title: نام افسانه
slug: yek-slug-یکتا
province: استان
city: شهر یا روستا
village: null
dialect: لهجه
creature: نوع موجود/دسته
tags: [برچسب۱, برچسب۲]
author: نام شما
created: 2026-07-01
---

متن کامل افسانه اینجا...
```

برای نسخه‌ی انگلیسی (اختیاری) کنارش `en.md` بگذار. اگر نگذاری، سایت اجباری نمی‌کند.

> پوشه‌ی legends باید همیشه فقط شامل متن باشد؛ هیچ کد یا فایل جاوااسکریپتی داخلش نگذار.

پس از افزودن فایل، فهرست را به‌روز کن:
```bash
node scripts/build-index.mjs
```
(در گیت‌هاب این کار هنگام push خودکار انجام می‌شود.)
